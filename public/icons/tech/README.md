# 🎨 Local Tech Stack Icons Directory

Place your custom icons in this directory (`public/icons/tech/`). 
The application is configured to load these icons locally.

### 📋 Expected Filenames & Mappings:

To ensure all icons display correctly, upload your `.svg` (recommended) or `.png` icons with the following exact filenames:

| Technology | Expected Filename | Recommended Format |
| :--- | :--- | :--- |
| **C++** | `cplusplus.svg` | SVG / PNG |
| **Python** | `python.svg` | SVG / PNG |
| **JavaScript** | `javascript.svg` | SVG / PNG |
| **TypeScript** | `typescript.svg` | SVG / PNG |
| **React** | `react.svg` | SVG / PNG |
| **Next.js** | `nextdotjs.svg` | SVG / PNG |
| **Node.js** | `nodedotjs.svg` | SVG / PNG |
| **Express** | `express.svg` | SVG / PNG |
| **MongoDB** | `mongodb.svg` | SVG / PNG |
| **SQL** | `mysql.svg` | SVG / PNG |
| **PostgreSQL** | `postgresql.svg` | SVG / PNG |
| **Tailwind CSS** | `tailwindcss.svg` | SVG / PNG |
| **Git** | `git.svg` | SVG / PNG |
| **Docker** | `docker.svg` | SVG / PNG |
| **AWS** | `amazonwebservices.svg` | SVG / PNG |
| **LangChain** | `langchain.svg` | SVG / PNG |

*Note: For **RAG**, the application renders a custom animated vector SVG natively in the code, so no local icon upload is required!*
